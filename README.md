# قالب سایت خدمات نظافتی

این یک **قالب سایت** طراحی شده برای شرکت‌های خدمات نظافتی است. قالب شامل ساختار، طراحی و ویژگی‌های پایه‌ای برای نمایش خدمات نظافتی به صورت آنلاین می‌باشد.

## ویژگی‌ها
- طراحی **مدرن و تمیز**  
- واکنش‌گرا (**Responsive**) برای موبایل و دسکتاپ  
- دکمه **تماس واتساپ شناور** برای ارتباط سریع با مشتری  
- آماده برای **سفارشی‌سازی** برای هر شرکت خدمات نظافتی  

## فایل‌ها
- `main.html` – فایل اصلی HTML سایت  
- `cleaning.png` – تصویر نمونه استفاده شده در قالب  
- سایر آیکون‌ها و فایل‌های کمکی برای استایل و عملکرد
